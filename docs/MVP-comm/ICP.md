5. Two types of riders exist: Contractual riders (what the venue provides) vs. Band riders (stage plot + input list + technical notes). Miked.live focuses on band riders â€” this is correct.
6. Power drops vs. Gear placement: Engineers care about electricity locations, not where a guitarist stands with a pedalboard. Simplify the stage plot mental model.
